# Car-sales-prediction

From the car sales data, develop an algorithm to predict the price of the cars based on various attributes associated with the car.
These attributes are - date of advertisment, car name, brand, model, nature of seller (private, commercial), offer type, abtest, vehicle type, year and month of registration,
gearbox, power of car, number of kilometers car has travelled, fuel type and status of repaired damages.

The packages used are Pandas, numpy seaborn and sklearn.
The algorithms used for prediction are Linear Regression and Random Forest.
